¿Cuál es la diferencia entre Internet y la World Wide Web?

La diferencia es que el internet es la interconexión entre las computadoras y la www es un servicio que funciona con el internet para acceder a páginas web

¿Cuáles son las partes de un URL?

Protocolo, Esquema, Dominio, Puerto, Ruta de acceso y Parametros de consulta

¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?

GET -> Obtiene datos del servidor (sin cifrar)

POST -> Obtiene datos del servidor (cifrado)

HEAD -> Estructura usada para los encabezados

PUT -> Remplaza datos de un recurso

PATCH -> Modifica parcialmente un recurso

DELETE -> Eliminar datos del servidor

¿Qué método HTTP se debe utilizar al enviar un formulario HTML, por ejemplo cuando ingresas tu usuario y contraseña en algún sitio? ¿Por qué?

POST ya que los datos se envian en el cuerpo de la solicitud y no en la URL, volviendolo más seguro

¿Qué método HTTP se utiliza cuando a través de un navegador web se accede a una página a través de un URL?

GET ya que se esta solo para enviar datos del navegador web

Un servidor web devuelve una respuesta HTTP con código 200. ¿Qué significa esto? ¿Ocurrió algún error?

Indica que una solicitud fue procesada con éxito, siendo uno de los códigos HTML más comunes y reconocibles

¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 404? ¿Por qué?

El código 404 se segmenta en dos partes: 4 (error del cliente causado por quien hizo la solicitud en la pagina al servidor) 04 (no encontrado, no se pudo completar la solicitud porque no se encontro el documneto HTML)

Si es responsabilidad del desarrollador corregir el error, porque puede que el error sea causado por por algun error en la ruta o recurso o se puede redirigir al usuario correctamente

¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 500? ¿Por qué?

Un error 500 significa que se ha producido un error al intentar acceder al servidor, pero no se puede dar más detalles sobre lo que ha ocurrido

Si es responsabilidad del desarrollador corregir el error, este error puede indicar problemas dentro del código o configuración del servidor y se debe de averiguar el origen del fallo

¿Cuál es el ciclo de desarrollo de sistemas de información?

Planificación

Consiste en un conjunto de actividades como:
Montar el proyecto
Justificar proyecto
Elegir la metodología de desarrollo
Desarrollar un cronograma del proyecto
Producir un plan de desarrollo del proyecto

Analisis

Se recopila información detallada sobre lo que debe de hacer el sistema, se consultan usuarios y partes interesadas para entender las funcionalidades necesarias

Diseño

Se crean los planos del sistema, como diagramas de flujo, arquitecturas de software y bases de datos

Implementación

En esta fase se hace el desarrollo del código y se desarrolla el sistema siguiendo el diseño previsto

Mantenimiento

Una vez implementado, el sistema necesita actualizaciones, correción de errores y mejoras para seguir funcionando eficiente
