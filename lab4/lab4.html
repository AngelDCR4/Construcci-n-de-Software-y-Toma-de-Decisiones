¿Qué diferencias y semejanzas hay entre Java y JavaScript?

Java se ejecuta en la JVM permitiendo usarlo en servidores, aplicaciones de escritorio, dispositivos moviles, etc. Mientras que JavaScript esta hecho principalmente para el desarrollo web

Las semejanzas estan entre la sintaxis inspirada en C y que ambos se pueden usar para lenguaje orientado a objetos

¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)

getFullYear() -> Da el año de la fecha

getMonth() -> Da el mes (0 para enero, 1 para febrero, ...)

getDate() -> Da el día del mes (1-31)

getDay() -> Da el día de la semana (0 para domingo, 1 para lunes, ...)

getTime() -> Da la marca del tiempo en milisegundos

¿Qué métodos tienen los arreglos? (Menciona al menos 5*)

push(elemento) -> Añade un elemento al arreglo

pop() -> Elimina y devuelve el ultimo elemento del arreglo

shift() -> Elimina y devuelve el primer elemento del arreglo

unshift(elemento) -> añade uno o más elementos al inicio del arreglo y devuelve la nueva longitud

map(callback) -> crea un nuevo arreglo con los resultados de la función callback aplicada a cada elemento

¿Cómo se declara una variable con alcance local dentro de una función?

Se puede hacer uso de let o const dentro de una función y al salir de la función sus valores ya no son accesibles

¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

Si se hace uso de variables globales es posible que la variable se haga uso en distintas partes del código, es una mala practica de encapsulamiento ya que rompe el principio y lo hace propenso a errores y reduce la modularidad del código

Es más recomendable hacer uso de variables locales o dentro de funciones como let y const para evitar estos riesgos
